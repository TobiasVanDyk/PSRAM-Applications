# PSRAM-Applications
A number of 64Mb (8MB) AP Memory Technology APS6404L-3SQN QSPI PSRAM had been obtained from Mouser (Texas) and is being tested as Single and Quad SPI memory interfaces with Teensy 4, 4.1 and ESP32's.
